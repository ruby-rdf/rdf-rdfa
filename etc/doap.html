Project description for RDF::RDFa.

RDF::RDFa is an RDFa reader/writer for Ruby using the RDF.rb library suite.

Implemented Specifications
  • RDFa in XHTML (RDFa Core 1.0)
  • RDFa Core 1.1
  • XHTML+RDFa 1.1
  • HTML+RDFa 1.1
  • RDFa Lite 1.1
  • Role Attribute
